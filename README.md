# My Portfolio

A modern, responsive portfolio website showcasing my web development projects and skills.

## Features

- Responsive design for all devices
- Portfolio showcase with project details
- Skills slider with animated counters
- Contact form integration
- Smooth scrolling and animations
- Modern UI/UX design

## Technologies Used

- HTML5
- CSS3 (SCSS)
- JavaScript (jQuery)
- Bootstrap 4
- Various JavaScript libraries (GSAP, ScrollMagic, Owl Carousel, etc.)

## Setup

1. Clone the repository
2. Open `index.html` in a web browser
3. For development, use a local server (XAMPP, WAMP, etc.)
